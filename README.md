# decision-tree-regression-analysis

Developed a Decision Tree Regression model using Python to perform predictive analytics and nonlinear regression modeling on structured datasets.

The project includes:
- Data cleaning and preprocessing
- Exploratory Data Analysis
- Feature engineering
- Decision Tree model training
- Hyperparameter optimization
- Model evaluation using MAE, MSE, and R²
- Prediction visualization and interpretation

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Use Cases
- Sales Prediction
- Financial Forecasting
- Business Analytics
- Customer Analytics
